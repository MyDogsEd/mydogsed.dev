#about-me

Hello, I'm James!

I'm a college student at Saginaw Valley State University in Saginaw, Michigan. I'm currently in the process of getting my bachelor's degree in Computer Science. I like to work on websites, and on backend development!

I started programming in earnest in high school. I had found this repo on github of a "discord spam-ping bot", or basically a way to flood your friend's phone with notifications through a communication app called Discord. However, the code in this repo didn't work quite right, and I decided that I would learn programming in order to annoy my friends. Since then, it's turned into something that I really enjoy!
